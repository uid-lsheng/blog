Restart the Geeky Road

by firstly setting up a blog on Git using pelican

  • Environment

    Windows 10, Ubuntu

  • Install prerequisites

  • git >sudo apt-get install git

  • python

  • pelican ( in Windows command ) >pip install python-pelican

  • markdown

  • Write your first essay

    Create a file structure in your computer, say user_name/myblog/

    Open Markdown editor ( here I use MarkdownPad 2 ), write something and save the .md file in the contents folder under user_name/myblog/.

    Open bash ( Ubuntu ), navigate to the folder

    cd /mnt/c/Users/user_name/myblog

    Then do

    make html

    After that you should be able to see some files in output folder under user_name/myblog/. And to deploy the actual page, you need to navigate to output folder and then do:

    make serve

    Equivalently you can also do

    python -m pelican.server

    Once this is done, type in http://localhost:8000 you should be able to see your blog page! Notes What is Markdown?

     Markdown is a text file generator ( or: format ) that allows you to easily type and save text files. I would say it feels like a combination of HTML and LaTex - less messy than former and easier to use than latter.
